# new_dog_old_tricks

A learning platform that teaches classic sales principles to modern start up young entrepreneur folks. 

This project exists to:
- Demonstrate full-stack engineering skills
- Show secure backend design (Java / Spring Boot)
- Practice deployment and operations
- Document technical decisions and tradeoffs

## Tech Stack (Planned )
- Frontend: Next.js, Tailwind CSS
- Backend: Java (Spring Boot)
- Database: PostgreSQL
- Auth: Session based with Spring Security
- Hosting: Vercel (frontend), Render (backend)

In active development
